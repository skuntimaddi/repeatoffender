# repeatoffender
Find the number of times a data element gets repeated in a set of data in a given window of data
This is a trick interview question
The inputs are a set of data elements. The number of elements are fixed at 25.
The value of each element is greater than 0 and less than 25
The idea of the program is to find the number of time a subset of elements are repeated.
